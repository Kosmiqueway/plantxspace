# Security Policy

## Reporting
- For vulnerabilities or security concerns, open a private security advisory on GitHub or email the maintainers listed in the repository.
- Please do not file public issues for undisclosed vulnerabilities.
- We aim to acknowledge reports within 5 business days.

## Scope
- Application code in this repository
- Content and configuration related to site deployment

## Exclusions
- Issues in third-party services or external links
- Social engineering attempts

## Expectations
- Provide reproduction steps and impact assessment when possible.
- Avoid proof-of-concept payloads that could harm environments; use minimal cases.
